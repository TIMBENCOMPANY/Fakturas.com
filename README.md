# Fakturas

### Modern AI-Powered Invoicing Platform

[Fakturas](https://fakturas.com) is an enterprise-grade, high-end automated billing and ledger platform designed for modern startups, digital agencies, and sovereign creative professionals. By combining fluid visual interfaces with offline-resilient natural language schema parsers, Fakturas streamlines invoice draft creation, international tax calculations, and secure ledger tracking into a single, cohesive workspace.

Visit the live platform at: **[fakturas.com](https://fakturas.com)**

---

## Technical Highlights & Features

- **Conversational Invoicing Core**: Instantly compile fully compliant, structured billing schemas from natural language inputs with intelligent fallback protection.
- **Dynamic Ledger Workspace**: A real-time sandboxed control ledger offering fluid tracking of accounts receivable, liquid assets, pending settlements, and live activities.
- **Multilingual & Multi-Style Templates**: Switch between premium design presets (Modern, Corporate, Freelancer, Agency) optimized for visual precision and typographic harmony.
- **Automated VAT & Tax Reports**: Ready-to-file logs covering EU One-Stop-Shop (MOSS) jurisdictions and US state sales tax nexuses.
- **Audit-Ready Cryptographic Trails**: Every invoice state transition registers an immutable, chronologically logged SHA-256 validation checksum.
- **First-Class Mobile Experience**: Fully responsive interface structured with high-contrast luxury gold elements and eye-safe twilight color palettes.

## Technology Stack

The platform is designed for peak speed, reliability, and security:

- **Frontend**: React 18+ with TypeScript, Vite, and [motion](https://github.com/motiondivision/motion) animations.
- **Styles**: Custom Utility-First Tailwind CSS with unified color system tokens.
- **Analytics**: Production-grade Vercel Web Analytics integration.
- **Packaging**: Standardized Node.js container with standalone production server capabilities.

## Development & Build Workflow

### Prerequisites
- Node.js LTS (v18+ recommended)
- npm or yarn

### Installation
Clone the repository and install all required modules:
```bash
npm install
```

### Dev Mode
Spin up the local development environment on port 3000:
```bash
npm run dev
```

### Standalone Production Build
Compile client-side bundles and package the custom Node server:
```bash
npm run build
npm start
```
